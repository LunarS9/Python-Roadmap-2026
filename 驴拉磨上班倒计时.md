
● 🫏  驴拉磨 · 项目文件结构

  donkey-mill/
  │
  ├── src-tauri/                          # Tauri 后端（Rust）
  │   ├── Cargo.toml                      # Rust 依赖声明与项目元信息
  │   ├── build.rs                        # Tauri 构建脚本
  │   ├── tauri.conf.json                 # 窗口配置：置顶、透明、无边框、尺寸、禁用音频
  │   ├── capabilities/
  │   │   └── default.json                # Tauri 2 权限声明（窗口控制、托盘、文件存取）
  │   ├── icons/
  │   │   └── icon.png                    # 应用图标主文件
  │   └── src/
  │       ├── main.rs                     # Tauri 入口：创建窗口、注册命令与托盘
  │       ├── tray.rs                     # 系统托盘：右键菜单（设置/暂停提醒/退出）
  │       ├── config.rs                   # 配置读写：序列化到 AppData 下的 JSON
  │       └── commands.rs                 # 前端可调用的 IPC 命令（读写配置、窗口拖拽）
  │
  ├── src/                                # 前端（渲染层）
  │   ├── index.html                      # 唯一页面骨架：挂载 Canvas 与 UI 覆盖层
  │   │
  │   ├── css/
  │   │   └── style.css                   # 全局样式：透明背景、像素字体、弹窗布局
  │   │
  │   ├── js/
  │   │   ├── main.js                     # 入口：初始化模块、启动渲染循环与定时器
  │   │   ├── countdown.js                # 倒计时引擎：计算当前时刻到下班的剩余秒数
  │   │   ├── reminder.js                 # 护眼提醒：30min 间隔触发视觉提示（纯静默）
  │   │   ├── renderer.js                 # Canvas 主循环：requestAnimationFrame 调度绘制
  │   │   ├── sprites.js                  # 精灵表加载器：解析 spritesheet 切片为帧序列
  │   │   ├── donkey.js                   # 驴精灵：行走帧动画、绕磨路径、状态切换
  │   │   ├── mill.js                     # 磨盘精灵：中心旋转帧动画、转速与剩余时间联动
  │   │   ├── particles.js               # 粒子系统：脚下扬尘与下班庆祝撒花
  │   │   ├── state.js                    # 全局状态机：正常/提醒/下班 三态管理
  │   │   └── config.js                   # 前端配置桥接：调用 Tauri IPC 读写用户设置
  │   │
  │   └── assets/
  │       ├── sprites/
  │       │   ├── donkey-walk.png         # 驴行走 spritesheet（侧面循环 8 帧，32×32/帧）
  │       │   ├── donkey-blink.png        # 驴停下眨眼 spritesheet（提醒态 4 帧）
  │       │   ├── donkey-celebrate.png    # 驴撒欢 spritesheet（下班态 6 帧）
  │       │   ├── mill.png                # 磨盘旋转 spritesheet（俯视 6 帧，64×64/帧）
  │       │   └── dust.png               # 扬尘粒子 spritesheet（4 帧，8×8/帧）
  │       └── fonts/
  │           └── pixel.ttf              # 像素点阵字体（倒计时数字渲染用）
  │
  ├── package.json                        # 前端依赖与 dev/build 脚本
  ├── .gitignore                          # 忽略 node_modules/target/dist
  └── README.md                           # 项目说明、构建方式、配置项文档

  约束备忘

  │      约束项      │                              执行方式                              │
  ├──────────────────┼────────────────────────────────────────────────────────────────────┤

  │ 透明无边框       │ tauri.conf.json → decorations: false + transparent: true           │
  ├──────────────────┼────────────────────────────────────────────────────────────────────┤
  │ 仅 Windows       │ 构建目标锁定 windows-x86_64；托盘行为遵循 Win 规范                 │
  ├──────────────────┼────────────────────────────────────────────────────────────────────┤
  │ 无外部运行时依赖 │ Vanilla JS，零 npm runtime 依赖，仅 devDependencies 含 Tauri CLI   │
  └──────────────────┴────────────────────────────────────────────────────────────────────┘
