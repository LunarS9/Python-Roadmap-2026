# 🫏 驴拉磨 · 项目文件结构

## 技术栈

- **桌面框架**：Tauri 2.x
- **前端**：Vanilla HTML/CSS/JavaScript + Canvas
- **动画风格**：Pixel Art
- **目标平台**：Windows x86_64
- **运行方式**：桌面悬浮窗 + 系统托盘

## 模块职责

### Rust（Tauri 后端）

| 文件 | 功能 |
|------|------|
| main.rs | 创建窗口、初始化应用 |
| tray.rs | 系统托盘与右键菜单 |
| config.rs | 用户配置读写 |
| commands.rs | IPC 通信接口 |

### Frontend（渲染层）

| 文件 | 功能 |
|------|------|
| main.js | 初始化与启动 |
| countdown.js | 工作时间计算 |
| reminder.js | 护眼提醒 |
| renderer.js | Canvas 主循环 |
| sprites.js | 精灵资源管理 |
| donkey.js | 驴动画控制 |
| mill.js | 磨盘动画控制 |
| particles.js | 粒子特效 |
| state.js | 状态机管理 |
| config.js | 配置同步 |

## 项目约束

| 约束项 | 实现方式 |
|--------|---------|
| 完全静音 | 不引入音频文件，仅视觉提醒 |
| 悬浮置顶 | `alwaysOnTop: true` |
| 透明无边框 | `transparent: true` + `decorations: false` |
| 仅 Windows | 构建目标锁定 `windows-x86_64` |
| 无运行时依赖 | Vanilla JS，无第三方运行时库 |
| 像素风 | 所有动画采用 SpriteSheet |

## 开发规范

### 文件新增规则

- 所有新增文件必须先更新本架构文档。
- 禁止跨目录直接引用资源。
- 所有状态变更统一经过 `state.js`。
- 所有配置读写统一经过 `config.rs` 与 `config.js`。
- Canvas 绘制统一由 `renderer.js` 调度。

### 命名规范

```text
JavaScript : camelCase
Rust       : snake_case
资源文件    : kebab-case
配置文件    : lower-case
```
